# منصة هتّاش التعليمية

نموذج تجريبي محلي لمنصة اختبارات تعليمية عربية مبنية باستخدام React وVite وTypeScript. يعمل المشروع دون قاعدة بيانات أو خدمات خارجية، ويحفظ تقدم الطالب ونتائجه محليًا عبر `localStorage`.

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح الرابط الذي يظهر في الطرفية.

## أوامر مفيدة

```bash
npm run typecheck
npm run build
npm run preview
```
